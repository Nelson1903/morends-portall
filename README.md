# Morends Portaal

Dit is het interne portaal voor de Morends B.V. Postbrievenbus- en Scanservice.

## Functies
- Veilige login (Supabase)
- Klantbeheer
- Uploaden en scannen van poststukken
- Automatische e-mail notificaties naar klanten
- Beveiligde opslag in de cloud

## Techniek
- React + Vite (frontend)
- Supabase (database + auth + storage)
- Vercel (hosting)
